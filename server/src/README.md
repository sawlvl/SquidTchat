# Source — C++/Qt

Ce dossier contient le code source du serveur WebSocket en C++ avec Qt.

## Lancer le projet

1. Ouvrir **Qt Creator**
2. Cliquer sur **Ouvrir un projet**
3. Sélectionner le fichier **SquidTchat.pro**
4. Configurer le projet en choisissant le kit de compilation
5. Lancer le serveur avec **Run** (`Ctrl+R`)
